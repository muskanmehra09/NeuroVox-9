# NeuroVox-9 
Voice Emotion Recognition using AI, ML 

##  Project Overview
NeuroVox‑9 is an intelligent system that recognizes human emotions from voice recordings using Artificial Intelligence (AI), Machine Learning (ML).  
Dataset: RAVDESS (Ryerson Audio‑Visual Database of Emotional Speech and Song).  
Features: Extracted using MFCC (Mel‑Frequency Cepstral Coefficients).  
Model: RandomForestClassifier.  
Accuracy: ~62.5% with confusion matrix evaluation.
