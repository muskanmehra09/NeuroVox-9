# NEUROVOX-9 🎤🤖

**AI-powered Emotion Recognition from Voice Samples**

---

## 📌 Project Overview
NEUROVOX-9 is a cutting-edge AI/ML project designed to decode human emotions from voice recordings.  
By combining **machine learning models** with a **Flask backend** and a **clean, professional frontend UI**, the system demonstrates how artificial intelligence can bridge technology and psychology.  

This project is not just a demo — it’s a showcase of end-to-end integration:  
- **Frontend** hosted live via GitHub Pages for accessibility.  
- **Backend** powered by Flask for real-time predictions.  
- **ML pipeline** trained to classify emotions such as happiness, sadness, anger, and neutrality.  

---

## ⚙️ Tech Stack
- **Backend:** Python, Flask, Librosa, Scikit-learn / TensorFlow  
- **Frontend:** HTML, CSS, JavaScript  
- **Deployment:** GitHub Pages (Frontend), Localhost (Backend)
